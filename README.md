# 🌸 BubblEase — Microdescansos para Desconectar

BubblEase es una experiencia web diseñada para quienes necesitan desconectar durante unos minutos del trabajo, los estudios o cualquier actividad que requiera concentración prolongada.

La aplicación combina pausas temporizadas, animaciones de respiración, mensajes motivadores con un toque de humor y una interfaz visual relajante. Su objetivo es transformar una pausa breve en una experiencia agradable, accesible y fácil de incorporar al día a día.

🌐 **Ver demo en producción**

👉 https://bubblease.vercel.app/

---

# ✨ Funcionalidades

## ⏳ Microdescansos Temporizados

* Descansos configurables de 1, 3 o 5 minutos.
* Temporizador con cuenta regresiva en tiempo real.
* Aviso automático al finalizar la sesión.

## 💬 Mensajes Motivadores

* Frases seleccionadas aleatoriamente.
* Rotación automática cada seis segundos.
* Enfoque desenfadado para generar una experiencia cercana.

## 🌊 Animaciones Relajantes

* Animación de respiración mediante pulsos y ondas.
* Partículas flotantes decorativas.
* Transiciones suaves entre estados.

## 🎵 Acceso a Música

* Acceso directo a Spotify para acompañar la pausa con música.

## 📱 Diseño Responsive

* Adaptación completa a móviles, tablets y escritorio.
* Interfaz optimizada para diferentes tamaños de pantalla.
* Experiencia consistente en cualquier dispositivo.

# 🚀 Instalación Local

No requiere dependencias externas ni proceso de compilación.

## 1. Clonar el repositorio

```bash
git clone https://github.com/BenitaPlata/BubblEase.git
```

## 2. Acceder al directorio

```bash
cd BubblEase
```

## 3. Ejecutar el proyecto

Abrir directamente:

```text
index.html
```

O iniciar un servidor local:

```bash
python -m http.server 8000
```

Abrir en el navegador:

```text
http://localhost:8000
```

También puede ejecutarse mediante extensiones como **Live Server** en Visual Studio Code.
